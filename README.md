# DVLD_Project
My first programming project in C# language


## Overview

The Driving & Vehicle License Department (DVLD) is a desktop application developed using **C#**, **.NET**, and **SQL Server**. The system automates the process of managing driving license services, applicants, drivers, and administrative operations.

The project follows a layered architecture that separates business logic from data access, making the application easier to maintain and extend.

---

## Features

### License Services

* Issue a new driving license.
* Renew an expired license.
* Replace a lost license.
* Replace a damaged license.
* Issue an international driving license.
* Detain a driving license.
* Release a detained license.

### Driving Tests

* Vision Test scheduling and management.
* Written Test scheduling and management.
* Practical Driving Test scheduling and management.
* Test retake management.
* Automatic validation of test sequence before issuing a license.

### People Management

* Register new people.
* Update personal information.
* Search using the national ID.
* Prevent duplicate records.

### Driver & License Management

* Issue licenses for different license classes.
* View license history.
* Validate license eligibility.
* Track license expiration dates.
* Manage active and historical licenses.

### User Management

* Create and manage system users.
* Edit user information.
* Enable or disable user accounts.
* Manage user permissions.

### Application Management

* Create and manage license applications.
* Track application status.
* View application history.
* Prevent duplicate active applications.
